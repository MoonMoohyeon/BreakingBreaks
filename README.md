# 보물 찾기(CHESS)

## 🗓️기간
2022.05.10. ~ 2025.06.03.

## ⚙️기술
- javascript

## 📖소개
웹 브라우저를 통해 플레이하는 벽돌 깨기 게임입니다.

## ✅기능
1. 게임 방법과 정보 소개
2. 방향키로 게임 진행
3. 환경 설정
4. 아이템
5. 순위표

## 👥역할
- 게임 콘셉트 및 스토리 구상
- 패들 및 벽돌 움직임 구현
- 게임 정보 및 점수 보기 구현
- 오디오 및 이미지 파일 연결

## 🚀실행
소스 코드를 웹 브라우저에서 실행합니다.
